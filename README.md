# Simulador de Pluriempleo e IRPF España

Frontend React + TypeScript para simular pluriempleo, retenciones y netos estimados.

## Motor fiscal

- Consolidación global anual del IRPF
- SS por empresa antes de IRPF
- Base liquidable con mínimo personal aplicado una sola vez
- Retenciones independientes por empresa
- Regularización estimada en renta

## Ejecutar

```bash
npm install
npm run dev
```

## Incluye

- Gestión de varios trabajos simultáneos
- Estimación de SS, IRPF y neto
- Timeline mensual de solapes
- Modo oscuro / claro
- Guardado local de escenarios
- Exportación JSON
